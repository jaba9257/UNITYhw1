Сделана полоскость она едет вместе с кубом и камерой по z координате, перед ней спаунятся 2 препядствия
1. высокий прямоугольник(2 урона)
2. низкий(1 урона)
когда они уезжают на 10 поинтов от куба исчезают, при столкновении исчезают
каждые 0.1 секунды ускоряется на 1%(можно изменить в инспекторе как в целом все)
все параметры заданы в скриптабл обджект, то есть генерируются только кубы
все параметры можно менять через инспектор
ускорение прописано как раз в x секунд объект ускоряется в x раз
